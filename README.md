# Relatório Redução dos Custos em Farmácias com AWS - Desafio de Projeto Santander Bootcamp 2025-2

O objetivo deste desafio era criar um relatório da análise para implementação de ferramentas AWS em uma empresa fictícia de farmácias, Abstergo Industries.  

## Processo

Foi fornecido um relatório modelo para a elaboração da versão final apresentada neste repositório. O projeto de implementação de ferramentas foi dividido em 3 etapas, cada uma com seus objetivos especí­ficos, os quais foram detalhados no relatório.

## Resultado

O relatório resultante da análise está apresentado no arquivo 'relatório.md', incluindo uma descrição inicial, as 3 etapas elaboradas com descritivo do caso de uso para cada uma, e uma conclusão.
